HTML & CSS Beginner's Guide

Let's Dive Into HTML & CSS!

HTML Tags (What Are These?)

<h1> to <h6>

These are your heading tags. They define titles or sections. <h1> is the biggest one, and <h6> is the smallest.

<h1>Biggest Heading!</h1>

<p>

It's for paragraphs of text. Very useful!

<p>This is a paragraph of text!</p>

<a>

This is for links. You can link to other pages or websites.

<a href="https://google.com">Click here to go to Google</a>

<div>

The <div> tag is a container. It doesn’t do much by itself, but you can use it to group things together.

<div>This is inside a div</div>

<img>

Want to add an image? Use the <img> tag! Just make sure to add a description too, with alt.

<img src="image.jpg" alt="An awesome image">

<ul> <ol>

<ul> makes unordered lists, and <ol> makes ordered (numbered) lists. They both use <li> for list items.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<span>

<span> is like a little helper. It’s used to style a small part of text.

<span>Just this part is styled!</span>

CSS Properties (Make It Pretty!)

font-family

This defines the font of the text. Use it to make things look good!

p {
    font-family: 'Arial', sans-serif;
}

color

Change the text color with this. Choose something that fits your style!

h1 {
    color: #333;
}

background-color

Change the background color of an element.

div {
    background-color: #f9f9f9;
}

padding

Padding is the space inside the element. So, it’s like the cushion around the content.

div {
    padding: 20px;
}

margin

This is the space outside the element. It's like giving it room to breathe.

div {
    margin: 15px;
}

border

Add a border around an element. You can choose the color and thickness!

div {
    border: 2px solid #333;
}

display

This controls how the element is displayed. Block or inline are the most common.

div {
    display: flex;
}

text-align

Align your text left, right, or center.

p {
    text-align: center;
}

font-size

Change the size of the font! Bigger text for big ideas.

p {
    font-size: 16px;
}
